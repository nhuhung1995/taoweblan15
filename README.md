# Global Internet 5G - V3

Phiên bản V3 được thiết kế theo hướng premium, ấm áp, 3D pseudo-depth và tối ưu cho việc trình bày với cấp quản lý hoặc khách hàng.

## Các trang chính
- Home
- Sản phẩm
- Pricing
- Giải pháp
- Hỗ trợ
- Liên hệ
- 5 trang con sản phẩm trong `/san-pham/*`

## Deploy
1. Upload toàn bộ project lên GitHub.
2. Trên Vercel, chọn Framework Preset là **Next.js**.
3. Nếu project nằm trong thư mục con, chọn đúng **Root Directory**.
4. Để trống **Output Directory**.
5. Deploy.
